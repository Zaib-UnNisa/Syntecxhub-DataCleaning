# Data Cleaning Utility

This project performs basic data cleaning operations on a student marks dataset.

## Features
- Handles missing values
- Fixes data types
- Removes duplicate records
- Standardizes column names
- Outputs cleaned dataset and cleaning log

## Tools Used
- Python
- Pandas
- Google Colab

## Dataset
Student study time, number of courses, and marks.
